Full-Stack Blog App

NarrativeNook is a dynamic blog application designed to provide a seamless and engaging user experience for both writers and readers. Built using React and Redux for a responsive and interactive frontend, and Node.js with Appwrite for robust backend support, NarrativeNook ensures efficient data management and secure authentication. The application features a clean and intuitive UI, styled with Tailwind CSS, making it easy for users to navigate and enjoy content. With Git for version control, the development process is streamlined and collaborative. NarrativeNook is the perfect platform for sharing stories, articles, and insights, fostering a vibrant community of bloggers and readers.

Currency Converter App

Currency Converter is a user-friendly application designed to provide real-time currency conversion. Built with React for a responsive and interactive interface, the app leverages external APIs to fetch the latest exchange rates, ensuring accurate and up-to-date conversions. The intuitive design, styled with Tailwind CSS, makes it easy for users to input amounts and select currencies for conversion. With no backend required, the app efficiently handles API calls directly from the frontend, offering a seamless and fast user experience. Ideal for travelers, business professionals, and anyone needing quick currency conversions, this app simplifies the process with just a few clicks.
